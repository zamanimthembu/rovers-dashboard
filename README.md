# Rovers Dashboard

An interactive rugby analytics dashboard built with React and Vite for exploring player KPIs, team metrics, and match performance data.

## Overview

Rovers Dashboard is a rugby performance and coaching dashboard focused on player evaluation, transparent team selection, and KPI-based feedback.

The dashboard is designed to support one-on-one coaching conversations by showing whether players are underperforming, meeting expectations, or excelling against defined targets. It turns coded post-match data into practical visual insight for coaches and players.

## Features

- Match dashboard view
- Player table with position filtering
- Selected player detail panel
- Team KPI summary
- Target vs actual comparisons
- Rugby-focused performance insights
- Instant visual feedback for coaching conversations

## Tech Stack

- React
- Vite
- JavaScript
- Material UI
- Recharts

## Data

The app uses structured JSON data derived from coded post-match source files.
